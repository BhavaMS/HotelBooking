# Hotel Booking Website

## Project Overview

The Hotel Booking Website is a web-based application developed using ASP.NET Core that allows users to browse hotels, view room details, and book rooms online. The system also provides separate functionalities for Customers, Hotel Administrators, and System Administrators.

## Features

### Customer Features

* Browse available hotels
* View room details and amenities
* Book hotel rooms
* Manage reservations

### Hotel Admin Features

* Add and manage hotel information
* Manage room availability
* View booking details

### System Admin Features

* Manage hotels and users
* Monitor bookings and system activities

## Technology Stack

* ASP.NET Core Web API
* C#
* Entity Framework Core
* SQLite Database
* HTML
* CSS
* JavaScript

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/BhavaMS/HotelBooking.git
   ```

2. Open the project in Visual Studio.

3. Restore NuGet packages.

4. Build and run the application.

5. Access the application through the local host URL.
